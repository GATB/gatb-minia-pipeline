Online Bacterial Genome Assembler

About

This tool enables the assembly of reads data using an online version of GATB-Minia-Pipeline.

GATB-Minia-Pipeline is a de novo assembly pipeline for Illumina data.

The pipeline consists of the automated execution of the following tools:

  • Bloocoo: error correction
  • Minia: contigs assembly
  • BESST: scaffolding

It can be used to assemble genomes and metagenomes. However, for this online version we limit its use to bacterial genomics, due to restriction on the amount of reads data you can transfer to our server. If you want to assemble larger genomes, please install and use the pipeline on your own computers. It is available from Github, here.

Pipeline default configuration

When using this online release of the GATB-Minia-Pipeline, the following configuration is used:

Tool Release Parameters
Bloocoo 1.0.7 -slow -high-precision
BESST 2.2.7 -iter 10000
Minia 3-prerelease defaults

Use

You can use the tools with two kind of files:

  • a single file containing all reads
  • two files containing paired-ended reads

In all cases, read files can be provided as FastA or FastQ; plain text or gzipped compressed files are accepted and we recommend you use gzipped files to reduce the amount of data to transfer from your computer to the remote server.


Request help, report issues

You are very welcome to contact us whether you have any questions regarding the use of this online service, as well as to report an issue. In all cases, please use the Issues system of the GATB-Assembly-Pipeline project hosted on Github.
